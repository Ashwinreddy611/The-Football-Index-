About

Who Are We?

The Football Index is a site that was built by fans, for the fans!. We obtain the essential data all fans want to know about and format it in the best way possible so you can get what you need quickly. Our passion is football, it always has been and always will be, which means we put our all into this site. So whether you wan to reminisce about the past or see what's going on now, we have you covered!

What's Our Future?

The Football Index is consistently thriving to be better and grow what we can provide our community. We aim to add more data and more genres of data for as many leagues to satisfy every football fans' every statistical desire. We aim to add some more surprise functionality with our map function so get in contact with us to set up an alert and tell us what you would like from The Football Index.
